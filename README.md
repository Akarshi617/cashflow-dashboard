# Cash-Flow Dashboard

Simple expense tracker made with plain HTML, CSS and JS. No frameworks, no libraries except Chart.js and jsPDF.

Add your salary, log expenses, see your balance update live. Data stays saved even after you close the tab.

## What it does

- Add salary and expenses
- Shows remaining balance instantly
- Saves everything in localStorage (refresh-proof)
- Delete any expense
- Pie chart for balance vs expenses
- Download your data as a PDF
- Warns you in red if balance gets too low
- Switch currency (INR/USD/EUR/GBP)

## Files

- `index.html`
- `style.css`
- `script.js`

## Run it

Just open `index.html` in your browser. That's it, no setup needed.
